프로젝트 실행 가이드


프로젝트 실행시 에러나는경우
1. 프로젝트 우클릭 -> 프로퍼티 -> 라이브러리 -> ojdbc6.jar 추가


db 설정방법(기본은 오라클로 설정했어요)
1. src -> main -> resources -> context-datasource.xml 수정


테이블 만드는 쿼리
1. document -> sql -> fs_1 쿼리로 테이블 생성하시면 되요!
